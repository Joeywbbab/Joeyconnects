# FlowTree 🌳

一个强大的原型交互流程管理工具，支持热点标注和并行步骤展示。

## 功能特性

- 📱 **交互流程管理** - 创建和管理原型交互流程
- 🔗 **跳转热点** - 添加可点击的热点，支持跳转到其他步骤
- 📝 **注释标识** - 添加注释标识区域，用于标注说明
- ⧉ **并行步骤** - 支持线性和并行两种布局模式
- 💾 **大容量存储** - 使用 IndexedDB，支持存储大量图片和数据
- 📤 **导入导出** - 支持 JSON 格式的导入导出，方便分享和备份
- 🎨 **高质量预览** - 保持原始图片质量，无压缩

## 使用方法

### Web App 模式（推荐）

1. 在浏览器中打开 `index.html`（主页面）
2. 点击 "FlowTree 编辑器" 卡片启动应用
3. 应用会在弹窗中打开，类似桌面小工具

### 直接访问

1. 直接在浏览器中打开 `app/app.html` 文件
2. 点击 "New Flow" 创建新的流程
3. 添加 Section、Group 和 Step
4. 上传截图并添加热点标注
5. 使用 "Preview" 预览交互流程

## 布局模式

- **线性模式 (Linear)** - 步骤按顺序排列，用箭头连接（默认）
- **并行模式 (Parallel)** - 步骤以网格形式展示，适合展示并行的用户路径

在 Group 页面点击布局切换按钮即可切换模式。

## 热点类型

- **🔗 跳转热点** - 点击可跳转到指定步骤
- **📝 注释标识** - 仅用于标注区域，不跳转

## 数据存储

- 使用 **IndexedDB** 存储，支持大容量数据
- 自动从 localStorage 迁移旧数据
- 支持导出 JSON 备份

## 技术栈

- 纯 HTML/CSS/JavaScript
- IndexedDB API
- 无需服务器，单文件即可运行
